# Выравнивание элементов с помощью FlexBox
Для выполнения задания, вам необходимо использовать свойства FlexBox:
* align-items
	* flex-start (по умол.)
	* flex-end
	* center
* justify-content
	* flex-start (по умол. )
	* flex-end
	* center
	* space-between
	* space-around
